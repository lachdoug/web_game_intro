Running HTML code

1

Create a plain text file on your computer called 'index.html'. Paste in this code:

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="utf-8"/>
  <title>Hello</title>
</head>

<style>
  h1 { color: blue; }
  h1.fade { opacity: 0.5; }
</style>

<body>
  <h1 id="heading">Hello world</h1>
</body>

<script>
  document.getElementById('heading').onclick =
  function() { this.className = "fade"; }
</script>

</html>

Save the file and load it in your browser.

2

Run the app in Codepen by pasting in the HTML, CSS and Javascript.

JSON

1

JSON: object data formatted as a string; used as a lightweight data-interchange format.

'{ "id": 4,
   "room_id": 2,
   "user": "Callie A",
   "body": "Is this data from the server?",
   "created_at": "2016-07-24T17:27:55.961Z" }'

You'll often see it with quotation marks escaped, looking like:

"{\"id\":4,\"room_id\":2,\"user\":\"Callie A\",\"body\":\"Is this data from the server?\",\"created_at\":\"2016-07-24T17:27:55.961Z\" }"

To extract data, parse JSON to create an object.

myJson = '{ "foo": "bar" }';
typeof(myJson);
myObject = JSON.parse( myJson );
typeof(myObject);
myObject.foo;
