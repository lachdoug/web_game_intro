HTML tag review

1

h1 - h6

p, br, div

a, strong, small, i, span

ol, ul, li

2

Build a website for your game (i.e. the class project game written in Scratch, Gamemaker or elsewhere).

Experiment with tags by adding content that describes your game, gives instructions on how to play, etc.

Here's a start. Open Codepen and paste this in a new pen:

<h1 id="page_title">My Game</h1>

<div class="body_text">
  <p>My Game is a cool Scratch app. Play it by pressing buttons and stuff.</p>

  <ul>
    <li>Press left/right arrow to move left/right.</li>
    <li>Up to go. Down to stop.</li>
  </ul>
</div>

<div class="footer">
  Share this game with friends...
</div>

body {
  font-family: Arial;
  font-size: 24px; }

#page_title { color: red; }

.body_text { color: #0000FF; }

.footer {
  text-align: center;
  background-color: #aaf; }
